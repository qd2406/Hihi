# Ô Ăn Quan 

Game dân gian Việt Nam dành cho điện thoại. Chơi **offline** (PvP / PvE với AI)

---


## Chạy ứng dụng

### 1. Cài dependencies

```bash
cd DesignGame
npm install
```
### 2. Cài công cụ tunnel (với máy mới)
```bash
npm install -g @expo/ngrok
```

### 3. Chạy Expo (offline mode)

```bash
npx expo start -c 
```

Quét QR bằng app **Expo Go** trên điện thoại.
